# WB Objection Trainer

AI-powered sales roleplay and drill trainer for Wealth Builders reps.

## Setup (2 minutes)

1. Install [Claude Code](https://claude.ai/download) if you don't have it
2. Open a terminal, navigate to this folder, and type `claude`
3. Say **"start"**

Claude does the rest. The only thing you'll need is a free API key from Google — Claude will tell you exactly where to get it.

## What you get

- Live voice roleplay with AI prospects
- Quick-fire objection drills with instant scoring
- Debrief analysis after each session
- Your scores and session history saved to your account

## Requirements

- Node.js 18+ (Claude will tell you if it's missing)
- Chrome or Edge browser (for voice features)
